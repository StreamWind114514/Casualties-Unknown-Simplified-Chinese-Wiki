## 补丁作用
> 你知道吗：翻译文件补丁文件全称 "Wiki CasualtiesUnknownDemo   chinese Chinese" ，缩减成两个字母便是"WC"，没错，我故意的wa(被打)

翻译覆盖范围（7.1版本）：

| 类别 | 数量 |
|------|------|
| 物品<br>(main) | 279 |
| 背景物块<br>(buildings) | 约70 |
| 状态效果<br>(moodles) | 171 |
| 液体种类<br>(other中的液体定义) | 74 |
| 生物种类<br>(buildings中的敌人生物) | 8 |
以及暂停界面文本、医疗界面文本、设置界面文本等多处地方

使用教程：下载好最新版的翻译补丁文件 (*WC.json*)，将该文件移动至`/CasualtiesUnknown_Data/Lang/`目录中，也就是游戏的语言文件夹"Lang"，接着进游戏主菜单，左键主菜单左边的空投罐，找到"语言"栏，选择显示为"维基中文v X.X"的翻译文本，鼠标移至上面即可显示介绍信息，点击后便可加载。
